## 개발환경
- Visual Studio Code

## 프로젝트 개요
- 부트스트랩을 사용하여 결제 시 웹디자인 틀 구현.
- open API를 사용하여 우편번호 클릭 시, 주소를 찾아서 누르면 자동으로 우편번호와 주소가 기입되어지게 구현.
- 카드선택 시, 누르는 카드 종류에 따라 다른 디자인과 그 카드 이름이 보여지게 구현.
- input 값의 내용이 없을 시, 결제하기 버튼을 눌렀을 때 alert가 팝업되어지게 구현.
